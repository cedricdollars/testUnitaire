## Exercices

Un site de e-commerce suit les achats effectués chaque jour. Le produit le plus acheté un jour est le produit vedette du jour suivant. S'il
y'a une égalité pour le produit acheté le plus fréquemment, ces noms de produits sont classés par ordre alphabétique croissant et le nom de
famille dans la liste est choisi.

Exemple
products = ['redShirt','greenPants','redShirt','orangeShoes','blackPants','blackPants']

- les pantalons verts et les chaussures oranges ont été acheté une fois
- les t-shirt rouges et les pantalons noirs ont été chacun acheté deux fois en un jour
- après avoir classé les produits par ordre alphabétique, le t-shirt rouge est le dernier produit dans la liste
- le t-shirt rouge est le produit en vedette pour le lendemain.

Compléter la fonction featuredProduct(products)
